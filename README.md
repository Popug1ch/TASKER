# 📋 Таскер — управление задачами, событиями и дедлайнами

Полнофункциональное веб-приложение для личного планирования с авторизацией, календарём и Telegram-ботом.

## 🌟 Особенности

- 🗓 **Календарь на неделю** с цветовыми метками: зелёные/жёлтые/красные/чёрные точки — приоритеты задач, синяя точка — события, оранжевая точка — дедлайны.
- ✅ **Задачи** — создание, редактирование, удаление, отметка выполнения.
- 🎉 **События** — дела на весь день (максимум 3 в день).
- 💣 **Дедлайны** — с визуальным «фитильком» и бомбой, который сгорает пропорционально времени.
- 🔔 **Уведомления в Telegram** о начале задач, приближающихся дедлайнах и событиях.
- 🔐 **Авторизация** через сессионные cookie (безопасные, HttpOnly).
- 🤖 **Telegram-бот** для просмотра задач, событий, дедлайнов и отметки выполненных.

## 🛠️ Технологии

| Компонент | Технология |
|-----------|------------|
| Бэкенд    | Python 3.11+, FastAPI, SQLAlchemy (async), SQLite |

## 🚀 Установка и запуск

1. Клонируйте репозиторий:  
   `git clone https://github.com/Popug1ch/TASKER.git && cd TASKER`
2. Создайте виртуальное окружение и активируйте:  
   `python -m venv .venv && source .venv/bin/activate` (Linux/Mac) или `.venv\Scripts\activate` (Windows)
3. Установите зависимости:  
   `pip install -r requirements.txt` (или вручную: `pip install fastapi uvicorn sqlalchemy aiosqlite python-telegram-bot python-dotenv passlib bcrypt schedule`)
4. Создайте `.env` с переменной `BOT_TOKEN=ваш_токен_от_BotFather`.
5. Запустите веб-приложение: `python main.py` — откройте http://127.0.0.1:8000
6. Запустите Telegram-бота (в отдельном терминале): `python -m bot.run`

## 🔐 Аутентификация

- **Регистрация:** `POST /auth/register` (username, email, password)
- **Вход:** `POST /auth/login` — устанавливается cookie `session_token` (HttpOnly, 30 дней)
- **Выход:** `POST /auth/logout`
- **Проверка:** `GET /auth/me` — возвращает данные пользователя

Все защищённые эндпоинты используют зависимость `get_current_user`, которая проверяет cookie и возвращает объект пользователя.

## 🌐 API (основные эндпоинты)

### Задачи (`/api/tasks`)
- `GET /` – все задачи пользователя (или за диапазон дат)
- `POST /` – создать задачу
- `PUT /{task_id}/status` – изменить статус
- `PUT /{task_id}` – обновить задачу
- `DELETE /{task_id}` – удалить задачу

### События (`/api/events`)
- `GET /all` – все события
- `POST /` – создать событие
- `GET /{event_date}` – события на дату
- `PUT /{event_id}` – обновить событие
- `DELETE /{event_id}` – удалить событие

### Дедлайны (`/api/deadlines`)
- `GET /all` – все дедлайны
- `GET /active` – только активные (невыполненные)
- `POST /` – создать дедлайн
- `GET /{deadline_date}` – дедлайны на дату
- `PUT /{deadline_id}/status` – изменить статус
- `PUT /{deadline_id}` – обновить дедлайн
- `DELETE /{deadline_id}` – удалить дедлайн

**Документация Swagger** доступна после запуска по адресу: `/docs` или `/redoc`.

## 🤖 Telegram-бот

### Команды
- `/start` – приветствие и главное меню
- `/register email пароль` – привязка Telegram к аккаунту
- `/tasks` – задачи на сегодня
- `/events` – события на сегодня
- `/deadlines` – дедлайны на ближайшие 24 часа
- `/complete` – отметить задачу выполненной (интерактивно)
- `/help` – справка

### Уведомления
Бот каждые 2 минуты проверяет БД и отправляет:
- Напоминания о задачах, начинающихся через 5–15 минут
- Предупреждения о дедлайнах за 24, 12, 3 и 1 час
- Утреннее уведомление о событиях на сегодня (около 9:00)

### Регистрация
1. Зарегистрируйтесь на сайте.
2. В боте выполните `/register ваш_email ваш_пароль`.
3. Telegram ID привяжется к профилю.
